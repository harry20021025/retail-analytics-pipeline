# 📖 Data Dictionary

## Dataset: Brazilian E-Commerce (Olist)
Source: Kaggle — https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
Time Period: 2016 — 2018
Total Orders: 96,000+

---

## 📦 olist_orders_dataset
Main orders table — one row per order

| Column | Type | Description |
|---|---|---|
| order_id | TEXT | Unique identifier for each order |
| customer_id | TEXT | Links to customers table |
| order_status | TEXT | delivered, cancelled, shipped etc |
| order_purchase_timestamp | DATETIME | When customer placed the order |
| order_approved_at | DATETIME | When payment was approved |
| order_delivered_carrier_date | DATETIME | When seller handed to carrier |
| order_delivered_customer_date | DATETIME | When customer received order |
| order_estimated_delivery_date | DATETIME | Estimated delivery date |

---

## 🛍️ olist_order_items_dataset
One row per item inside each order

| Column | Type | Description |
|---|---|---|
| order_id | TEXT | Links to orders table |
| order_item_id | INT | Item sequence number within order |
| product_id | TEXT | Links to products table |
| seller_id | TEXT | Links to sellers table |
| shipping_limit_date | DATETIME | Deadline for seller to ship |
| price | DECIMAL | Product price in Brazilian Real (R$) |
| freight_value | DECIMAL | Shipping cost in R$ |

---

## 👥 olist_customers_dataset
One row per customer

| Column | Type | Description |
|---|---|---|
| customer_id | TEXT | Unique customer ID (links to orders) |
| customer_unique_id | TEXT | Permanent customer identifier |
| customer_zip_code_prefix | TEXT | Customer zip code |
| customer_city | TEXT | Customer city |
| customer_state | TEXT | Customer state (SP, RJ, MG etc) |

---

## 📦 olist_products_dataset
One row per product

| Column | Type | Description |
|---|---|---|
| product_id | TEXT | Unique product identifier |
| product_category_name | TEXT | Category in Portuguese |
| product_name_lenght | INT | Number of characters in name |
| product_description_lenght | INT | Number of characters in description |
| product_photos_qty | INT | Number of product photos |
| product_weight_g | INT | Weight in grams |
| product_length_cm | INT | Length in centimeters |
| product_height_cm | INT | Height in centimeters |
| product_width_cm | INT | Width in centimeters |

---

## ⭐ olist_order_reviews_dataset
Customer reviews for each order

| Column | Type | Description |
|---|---|---|
| review_id | TEXT | Unique review identifier |
| order_id | TEXT | Links to orders table |
| review_score | INT | Rating 1-5 (5 = best) |
| review_comment_title | TEXT | Review title |
| review_comment_message | TEXT | Full review text |
| review_creation_date | DATETIME | When review was created |
| review_answer_timestamp | DATETIME | When seller responded |

---

## 💳 olist_order_payments_dataset
Payment details for each order

| Column | Type | Description |
|---|---|---|
| order_id | TEXT | Links to orders table |
| payment_sequential | INT | Payment sequence number |
| payment_type | TEXT | credit_card, boleto, voucher, debit_card |
| payment_installments | INT | Number of installments chosen |
| payment_value | DECIMAL | Amount paid in R$ |

---

## 🏪 olist_sellers_dataset
Seller information

| Column | Type | Description |
|---|---|---|
| seller_id | TEXT | Unique seller identifier |
| seller_zip_code_prefix | TEXT | Seller zip code |
| seller_city | TEXT | Seller city |
| seller_state | TEXT | Seller state |

---

## 🌐 product_category_name_translation
Translates Portuguese category names to English

| Column | Type | Description |
|---|---|---|
| product_category_name | TEXT | Category name in Portuguese |
| product_category_name_english | TEXT | Category name in English |

---

## 🤖 rfm_segmented (Generated by Python)
Output of KMeans clustering — one row per customer

| Column | Type | Description |
|---|---|---|
| customer_id | TEXT | Links to customers table |
| last_purchase_date | DATETIME | Date of most recent order |
| recency_days | INT | Days since last purchase |
| frequency | INT | Total number of orders placed |
| monetary | DECIMAL | Total amount spent in R$ |
| Cluster | INT | KMeans cluster number (0-3) |
| Segment | TEXT | Champions, Recent, At Risk, Lost |
